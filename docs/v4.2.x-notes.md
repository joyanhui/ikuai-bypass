# v4.2.x 版本注意事项

## 版本兼容性

- **v4.2.x 兼容爱快 3.x.x**
- **不兼容爱快 v4.x**（爱快 4.x API 变更较大，请使用 Rust 主线版本 v4.4.100+）

> 老用户从 v4.2.x 升级到 Rust 主线版本请先阅读[升级指南](v4.4.13-update-to-v4.4.10x)。

## WebUI 在线保存配置（默认禁用）

v4.2.x 的 WebUI **默认禁用**了配置文件在线保存功能，以防止误操作覆盖本地配置。

如需启用，需手动编辑 `config.yml`：

```yaml
webui:
  enable-update: true  # 是否启用配置文件在线更新功能
```

- 默认值：`false`（禁用）
- 设为 `true` 开启后，WebUI 的保存按钮才可用
- 禁用时：前端保存按钮置灰，并显示"在线更新已禁用"的红色警告横幅；后端返回 403 Forbidden

> 该配置项仅存在于 v4.2.x Go 版本中。Rust 主线版本已移除该开关，始终强制开启在线保存功能。
