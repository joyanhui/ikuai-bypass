# v4.2.x 版本注意事项

## 版本兼容性

- **v4.2.x 兼容爱快 3.x.x** v4.2.0是最后一个兼容爱快3.x的稳定版本
- **不兼容爱快 v4.x**（爱快 4.x 最新版请使用最新版 ikuai-bypass）
- v4.2.0 及之前版本未提供 Docker 镜像，如需在爱快 3.7 中使用 Docker 部署请参考 [#112](https://github.com/joyanhui/ikuai-bypass/discussions/112)

> 老用户从 v4.2.x 升级到 Rust 主线版本请先阅读[升级指南]({{ '/v4.4.13-update-to-v4.4.10x/' | relative_url }})。

## WebUI 在线保存配置（默认禁用）

v4.2.x 的 WebUI **默认禁用**了配置文件在线保存功能，以防止误操作覆盖本地配置。

如需启用，需手动编辑 `config.yml`：

```yaml
webui:
  enable-update: true  # 是否启用配置文件在线更新功能
```

- 默认值：`false`（禁用）
- 设为 `true` 开启后，WebUI 的保存功能才可用
- 禁用时：会报错 403 Forbidden错误

> 该配置项仅存在于 v4.2.x Go 版本中。大于4.4.0的版本已移除该开关，始终强制开启在线保存功能。
